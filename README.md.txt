# Even or Odd Checker 🔢

A simple C program that checks whether a number entered by the user is even or odd.

---

## 💻 Project Description

This is a basic C programming project developed as part of our Structured Programming Lab course. The program takes an integer input from the user and prints whether the number is even or odd.

## 🛠️ How It Works

1. The user is prompted to enter an integer.
2. The program checks if the number is divisible by 2 using the modulo operator (`%`).
3. If divisible by 2 → it's even. Otherwise → it's odd.
